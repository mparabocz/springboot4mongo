# SpringBoot4Mongo

This project was developed as part of my studies in Spring Boot and MongoDB.  
It demonstrates how to build a RESTful API using a NoSQL database, following clean architecture principles.

## Project Goals

- Practice building APIs with Spring Boot
- Integrate a NoSQL database (MongoDB)
- Understand document-based data modeling
- Implement CRUD operations using Spring Data
- Explore repository pattern with MongoDB

## Technologies Used

- Java 25
- Spring Boot 4
- Spring Data MongoDB
- MongoDB
- Maven

## Features

- RESTful API structure
- CRUD operations (Create, Read, Update, Delete)
- Repository layer using Spring Data
- Document-based persistence with MongoDB
- Clean separation of layers (Controller, Service, Repository)
